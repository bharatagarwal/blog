LLDB Abbreviations Command Description add-dsym Add a debug symbol file to one of the target’s current modules by specifying a path to a debug symbols file or by using the options to specify a module. attach Attach to process by ID or name. b Set a breakpoint using one of several shorthand formats. bt Show the current thread’s call stack. Any numeric argument displays at most that many frames. The argument ‘all’ displays all threads.
